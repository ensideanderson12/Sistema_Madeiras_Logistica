# Sistema Madeiras Logística

## 🌲 Sistema de Transparência Total - Operação Curitiba → Rio de Janeiro

### Visão Geral
Sistema completo para análise estratégica da operação logística **Mandirituba/PR → Rio Pomba/MG** com modelo disruptivo de transparência total, autonomia do produtor e divisão justa do lucro excedente.

### 📊 Dashboard Principal
Acesse: `index.html`

**Funcionalidades:**
- Dashboard em tempo real da operação Curitiba-RJ
- Análise de produtos (Pinus Convencional + Seco)
- Métricas de transparência e lucratividade
- Comparativo modelo tradicional vs transparente

### 🧮 Calculadora de Transparência
**Características:**
- Simulação completa de custos e lucros
- Análise de ROI para fornecedor e sistema
- Recomendações estratégicas automáticas
- Exportação para Excel e relatórios PDF

### 🚛 Módulo Logística Inteligente
**Recursos:**
- Análise de rotas alternativas
- Comparação de transportadoras qualificadas
- Otimização de custos e tempo
- Análise preditiva de mercado

### 📈 Sistema Comparativo
**Análises:**
- Modelo tradicional vs transparente
- Diferentes rotas Sul-Sudeste
- Variações de produtos (Convencional vs Seco)
- Impacto da distância nos custos

## 🎯 Dados da Operação Base

### Rota Logística
- **Origem:** Mandirituba/PR
- **Destino:** Rio Pomba/MG
- **Distância:** 1.319 km
- **Frete Contratado:** R$ 11.000,00
- **Custo por km:** R$ 8,33 (21% abaixo da média nacional)

### Produtos Analisados
1. **Pinus Convencional**
   - Volume: 28,000 m³
   - Dimensões: 2,0 x 30 x 3,00
   - Preço: R$ 1.170,00/m³
   - Valor Total: R$ 32.760,00

2. **Pinus Seco**
   - Volume: 24,000 m³ (7+7+5+5 m³)
   - Preço: R$ 950,00/m³
   - Valor Total: R$ 22.800,00

### Métricas Calculadas
- **Volume Total:** 52 m³
- **Peso Médio:** 500 kg/m³
- **Preço Médio Colocado:** R$ 1.280,00/m³
- **Custo Frete por m³:** R$ 211,54

## 💰 Modelo de Transparência Total

### Princípios
1. **Transparência Absoluta:** Todos os custos são compartilhados
2. **Autonomia do Produtor:** Decisões independentes mantidas
3. **Divisão Justa:** 70% fornecedor / 30% sistema do lucro excedente
4. **Sustentabilidade:** Relacionamento de longo prazo

### Benefícios Comprovados
- **Fornecedor:** +23% de receita vs modelo tradicional
- **Cliente:** Preços 15% mais competitivos
- **Sistema:** Relacionamentos duradouros e crescimento sustentável
