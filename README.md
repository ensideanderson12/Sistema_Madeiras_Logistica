# Sistema Madeiras Logística - PowerShell Optimization

## 🎯 Visão Geral

Este projeto implementa uma solução completa de otimização estratégica para o Sistema Madeiras Logística, incluindo um script PowerShell robusto e um dashboard web de transparência total com modelo financeiro 70/30.

## 🚀 Funcionalidades Implementadas

### Script PowerShell de Otimização (`scripts/otimizacao-completa.ps1`)
- ✅ **Validação completa** de integridade do sistema
- ✅ **Backup automático** antes de modificações
- ✅ **Minificação automática** de CSS e JavaScript
- ✅ **Criação do sistema** de transparência
- ✅ **Logs detalhados** de todas as operações
- ✅ **Modo de teste** para validação segura

### Dashboard de Transparência
- ✅ **KPIs em tempo real** (Receita, Operações, ROI, Combustível)
- ✅ **Calculadora 70/30** com validação robusta
- ✅ **Interface responsiva** para desktop e mobile
- ✅ **Breakdown detalhado** de custos operacionais
- ✅ **Sistema de salvamento** de cenários
- ✅ **Exportação de relatórios**

### Módulos Avançados
- ✅ **IA de Otimização** com algoritmos genéticos
- ✅ **Integração de Mercado** com dados de combustível
- ✅ **PWA Service Worker** para funcionalidade offline
- ✅ **Gráficos interativos** com fallback simples

## 📋 Como Usar

### 1. Executar Script de Otimização

```powershell
# Execução completa
.\scripts\otimizacao-completa.ps1

# Modo de teste (sem alterações)
.\scripts\otimizacao-completa.ps1 -TestMode

# Pular backup
.\scripts\otimizacao-completa.ps1 -SkipBackup
```

### 2. Acessar Dashboard

1. Abra `dashboard.html` em um navegador web
2. Navegue pelas seções usando o menu superior
3. Use a calculadora para análises de transparência
4. Monitore KPIs em tempo real

### 3. Calculadora de Transparência

1. **Preencha os campos:**
   - Distância da Rota (km)
   - Preço Combustível (R$/L)
   - Peso da Carga (ton)
   - Preço da Madeira (R$/m³)

2. **Clique em "Calcular Transparência"**

3. **Analise os resultados:**
   - Custo Total da Operação
   - Receita Bruta
   - Margem Operacional (70%)
   - Reserva Estratégica (30%)
   - ROI Projetado
