## 🚀 Guia de Início Rápido

### Acessando o Sistema
1. Abra o arquivo `index.html` em um navegador moderno (Chrome, Firefox, Safari, Edge)
2. O sistema carregará automaticamente com os dados da operação Curitiba-RJ
3. Use o menu superior para navegar entre as funcionalidades

## 🧮 Calculadora de Transparência

### Como Usar
1. **Ajuste os parâmetros** da operação nos campos disponíveis:
   - Volume do fornecedor (m³)
   - Preço base por m³
   - Distância da rota
   - Custo do frete
   - Margem transparente (%)
   - Divisão de lucro fornecedor (%)

2. **Clique em "Calcular Transparência"** para ver os resultados

3. **Analise os resultados** detalhados:
   - Valores financeiros completos
   - ROI e margens
   - Recomendações estratégicas
   - Comparativo modelo tradicional vs transparente

### Validações Automáticas
- **Volume**: Entre 0,1 e 100 m³
- **Preço**: Entre R$ 500 e R$ 3.000/m³
- **Distância**: Entre 50 e 3.000 km
- **Frete**: Entre R$ 1.000 e R$ 50.000
- **Margem**: Entre 5% e 30%
- **Divisão**: Entre 50% e 80%

### Funcionalidades Avançadas
- **Salvar Cenários**: Preserve configurações para comparação futura
- **Exportar Relatórios**: Gere PDFs ou Excel das análises
- **Comparações**: Veja vantagens do modelo transparente

## 🛠️ Solução de Problemas

### Problemas Comuns

#### O sistema não carrega
- **Verifique conexão** com internet (para fontes CSS/JS)
- **Use navegador atualizado** (Chrome 90+, Firefox 88+, Safari 14+)
- **Desabilite bloqueadores** de conteúdo temporariamente

#### Calculadora não funciona
- **Verifique valores inseridos** dentro dos limites
- **Limpe cache** do navegador (Ctrl+F5)
- **Recarregue a página** completamente

#### Dados não salvam
- **Verifique permissões** do navegador para localStorage
- **Use modo privado** como teste alternativo
- **Exporte manualmente** dados importantes

### Navegadores Recomendados
- ✅ **Google Chrome** 90+ (Recomendado)
- ✅ **Mozilla Firefox** 88+
- ✅ **Safari** 14+
- ✅ **Microsoft Edge** 90+
- ❌ Internet Explorer (não suportado)
